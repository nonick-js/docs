# イベントログ
## 概要
イベントログを使用すると、サーバー内でタイムアウトやBan、Kickなどの特定のイベントが発生した際にそのログを送信するようになります。
監査ログよりもわかりやすく、また通知を受け取ることもできるため、複数人でサーバーを管理している際に便利です。

![](/static/features/log/1.png)

## 対応しているイベント
- [x] Timeout
- [x] Ban
- [x] Kick
- [x] VCの接続・切断・移動
- [x] メッセージ削除

!!!
`Timeout`イベントは、AutoModの自動ブロックによるものの場合は送信されません。
!!!

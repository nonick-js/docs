# /setting

!!!success
このコマンドの実行には`サーバー管理`権限が必要です。
!!!

NoNICK.jsのコントロールパネルを開くコマンドです。コントロールパネルは、NoNICK.jsに搭載されている一部機能のカスタマイズをすることができます。

!!!
コントロールパネルの使い方に関する情報は、[NoNICK.jsの設定](/NoNICK.js/setting.md)を参照ください。
!!!
